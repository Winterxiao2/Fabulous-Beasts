### Chiese(中文)
## 图片命名规范
1. 图片在可以描述的情况下，请使用英文进行简单描述。
2. 如果知道作者及其出处，请一并标明！【具体可参考已命名的文件】
3. 未知作者与出处请忽略。
4. 如果图片内容难以描述，请使用图中角色名+"Unname"【角色较多请直接使用"Unname"】
5. 若图片为组图，请在描述后编号！
### 英文(English)
## Image Naming Guidelines
1. If the image content is describable, please use a brief English description for the filename.
2. If the author and source are known, please include them in the filename! 【Refer to existing named files for examples】
3. Omit the author and source information if they are unknown.
4. If the image content is hard to describe, use the character name(s) in the image followed by "Unname" 【Use only "Unname" if there are multiple characters】
5. If the image is part of a set, add a sequence number after the description!
